# Implementación de Estándares W3C

## Aplicación de los Estándares
En este proyecto, se han aplicado los siguientes estándares:

### 1. **HTML5**
- Uso de etiquetas semánticas como `<header>`, `<section>`, `<article>`, `<footer>` para estructurar el contenido de manera lógica y accesible.
- Inclusión de metadatos en el `<head>` para mejorar la compatibilidad y el SEO.
- Implementación de un diseño responsivo mediante el atributo `viewport`.

### 2. **CSS3**
- Uso de **CSS Grid** para crear un diseño de cuadrícula en la sección de estándares principales.
- Uso de **media queries** para garantizar que el diseño sea adaptable a diferentes tamaños de pantalla.
