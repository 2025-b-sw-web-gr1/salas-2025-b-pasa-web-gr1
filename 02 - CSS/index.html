Formas de cargar estilos en HTML

Proyecto educativo: ejemplos y explicación.

1) Inline CSS (estilo en atributo "style")

Aplicado directamente en el atributo style de un elemento.

Este bloque usa el atributo style.

2) Estilos inline con la propiedad style de DOM (JS)

Modificando estilos desde JavaScript usando element.style.

Usa JS para cambiar mi color en 1s...

3) Estilo interno: <style> en el documento

Se coloca en el <head> o dentro del <body> y aplica a todo el documento.

Este bloque usa una etiqueta <style> interna.

4) Hoja externa con <link rel="stylesheet">

La forma más común: archivo CSS servido por HTTP.

Este bloque usa la hoja externa css/external.css.

5) @import en CSS

Se puede importar otra hoja desde una hoja CSS usando @import. Atención al rendimiento: puede bloquear renderizado cuando se usa en el head.

Este bloque muestra estilos importados desde otra hoja CSS.

6) Preload de CSS

Permite priorizar la carga de una hoja (se convierte en rel=stylesheet cuando carga). Útil para performance.

Este bloque demuestra estilos que podrían precargarse.

7) Carga dinámica con JavaScript (<link>)

Crear y añadir un <link> dinámicamente desde JS.

Estilos dinámicos via <link>

8) Fetch + inject (CSS inline vía <style>)

Traer el contenido CSS por HTTP y inyectarlo en un <style>. Útil para manipulación o pequeñas reglas.

Estilos añadidos por fetch + inject.

9) Shadow DOM (estilos encapsulados)

Estilos aplicados a un Shadow Root no afectan el documento y viceversa.

10) CSS Modules / Bundlers (nota)

En aplicaciones modernas (React, Vite, Webpack) existen soluciones que generan clases únicas. No se muestra un build aquí, pero es importante conocerlo.

Revisa js/dynamic-styles.js para ver cómo funciona la carga dinámica y css/external.css y css/imported.css para ejemplos de reglas.
